# Ansible module: ansible.module_aws_acm_facts


Retrieve certificate facts from AWS Certificate Manager service

## Description

Retrieve facts for ACM certificates

## Examples


``` yaml

- name: obtain all ACM certificates
  aws_acm_facts:

- name: obtain all facts for a single ACM certificate
  aws_acm_facts:
    domain_name: "*.example_com"

- name: obtain all certificates pending validiation
  aws_acm_facts:
    statuses:
    - PENDING_VALIDATION

```
