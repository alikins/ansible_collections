# Ansible module: ansible.module_avi_stringgroup


Module for setup of StringGroup Avi RESTful Object

## Description

This module is used to configure StringGroup object
more examples at U(https://github.com/avinetworks/devops)

## Examples


``` yaml

  - name: Create a string group configuration
    avi_stringgroup:
      controller: '{{ controller }}'
      password: '{{ password }}'
      username: '{{ username }}'
      kv:
      - key: text/html
      - key: text/xml
      - key: text/plain
      - key: text/css
      - key: text/javascript
      - key: application/javascript
      - key: application/x-javascript
      - key: application/xml
      - key: application/pdf
      name: System-Compressible-Content-Types
      tenant_ref: admin
      type: SG_TYPE_STRING

```
